A strict macOS utility that enforces attention management by cutting off distracting applications.

// Core Directive

FocusOverlay is a lightweight macOS menu bar app. It keeps a searchable list of installed applications, lets you mark any as blocked, and throws a full-screen overlay on top of them when they become frontmost.

What it does

Capabilities & Protocols

01

System

  • Runs as a menu bar extra (LSUIElement) instead of a normal dock app
  • Indexes standard application directories automatically
  • Persists blocked bundle IDs securely in system UserDefaults
02

Control

  • Real-time search, filter, block, unblock capabilities
  • Bulk-update functionality for rapid visibility status changes
  • Configurable Pomodoro methodology (Focus, Breaks)
03

Enforcement

  • Full-screen visual overlay deployed on every connected display
  • Timer state is directly reflected on the interruption overlay
  • Forces an absolute choice: Quit or Jump back instantly

How to use it

  • 01. Launch FocusOverlay.
  • 02. Open the menu bar icon.
  • 03. Search for an app by name or bundle ID.
  • 04. Toggle apps into the blocked list.
  • 05. Start the timer for visual enforcement.
  • 06. Switch into a blocked app to trigger the overlay.

Implementation Notes

Blocked apps are identified purely by their Bundle ID. The installed-app index scans top-level .app bundles in standard application folders.

The overlay utilizes borderless windows at the .screenSaver level, ensuring it sits structurally above all regular application windows.

Current Limitations

This project focuses entirely on interruption, not hard system-level enforcement. A blocked app can still be reopened unless you keep it on the blocked list and let the overlay catch it again.
